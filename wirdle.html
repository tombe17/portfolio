Wirdle Website

The Project

For my web development class we were given the option to make our own website. I decide to create a recreation of the game wordle. I started with HTML, CSS, and JavaScript. We then added in a database, websocket, and APIs.



User Testing

I originally had a button under the board to reset the game. After testing with users, many of them clicked it thinking it was a submit button. I changed the design to have it only display a submit button until you complete the game.



The Resolution

By having users test my application, I was able to design it for them and consider how my application would actually be used by others.
